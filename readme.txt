The simple. When you post new article or update, delete nginx cache to reflect changes.

== Description ==
This plugin is delete Nginx cache.
To delete cache files by saving dir path.


== Installation ==

1. Upload deleteNginxCache directory to the `/wp-content/plugins/`
2. Activate the plugin through the ‘Plugins’ menu in WordPress
3. Setting menu is Side bar. Add path to nginx cache dir and save.
